# 安全策略

## Supported Versions / 支持范围

This repository is under active development. Only the latest commit on the default branch is supported.

本仓库处于持续开发中，仅默认分支最新版本提供支持。

## Reporting a Vulnerability / 漏洞报告

**EN**
- Do **not** open a public issue for sensitive reports.
- Use GitHub Security Advisories (preferred) or contact the maintainer via a private channel.
- Please include minimal reproduction steps and impact.

**中文**
- 涉及敏感安全问题请 **不要** 直接开公开 Issue。
- 建议使用 GitHub Security Advisories（优先），或通过私密渠道联系维护者。
- 请提供最小复现步骤与影响说明。

## Secrets & Data / 密钥与数据

- Never commit API keys, tokens, personal data, logs, screenshots, databases, or model files.
- Local config file `config/settings.json` is ignored by git.
