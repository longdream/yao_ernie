# Dependency Notes
# ==========================================
# langchain-core - Core LangChain framework for LLM orchestration
# langchain-openai - OpenAI integration for LangChain (supports any OpenAI-compatible API)
# pydantic      - Data validation and settings management
# json-repair   - Fix and parse incomplete JSON from LLM output
# nest-asyncio  - Support for nested async event loops (Jupyter)
# psutil        - System and process management
# httpx         - Async HTTP client for API calls
# requests      - Sync HTTP client (fallback)
# numpy         - Numerical computation (vector operations)
# tqdm          - Progress bar display for long tasks
# toml          - TOML configuration file parsing
# openai        - OpenAI API client (required by AgentScope)
# PyPDF2        - PDF document processing
# python-docx   - Word document processing
# jieba         - Chinese word segmentation
# rank-bm25     - BM25 text retrieval algorithm
# pywinauto     - Windows UI automation
# pyautogui     - Cross-platform GUI automation
# Pillow        - Image processing
# imagehash     - Image similarity detection
