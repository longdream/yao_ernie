# Code of Conduct
#
# 行为准则

We follow the Contributor Covenant Code of Conduct.

我们遵循 Contributor Covenant 行为准则。

## Our Pledge / 承诺

We pledge to make participation in our community a harassment-free experience for everyone.

我们承诺为所有参与者提供一个无骚扰的社区环境。

## Our Standards / 标准

Examples of behavior that contributes to a positive environment include:
- Being respectful and considerate
- Accepting constructive feedback
- Focusing on what is best for the community

Examples of unacceptable behavior include:
- Harassment or discrimination
- Trolling, insulting/derogatory comments
- Publishing others' private information

## Enforcement / 执行

Project maintainers are responsible for clarifying standards and enforcing this code of conduct.

If you experience or witness unacceptable behavior, please report it via the process in `SECURITY.md`.
